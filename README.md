# 📊 Bakliwal Tutorials - JEE Student Marks Dashboard

A comprehensive marks dashboard for JEE 2027 students to track their performance across multiple tests.

## Features
- 📈 Interactive performance charts
- 📊 Subject-wise analysis
- 📋 Complete test history
- 🏆 Best rank tracking
- 📥 Download results as CSV

## Technology Stack
- Streamlit
- Plotly
- Pandas
- Google Sheets API integration

## Deployment
This app is deployed on Streamlit Cloud and integrates with:
- Google Sheets for data storage
- PythonAnywhere Flask app for SSO
- Google Apps Script API for data fetching
